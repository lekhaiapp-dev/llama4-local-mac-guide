# System Requirements for Running Llama 4 on Mac

Before installing Llama 4 locally, make sure your Mac meets the following requirements.

## Processor

- Apple M1
- Apple M2
- Apple M3
- Apple M4

## RAM

### 8GB

Suitable for Scout models.

### 16GB

Recommended for most users.

### 32GB+

Ideal for Maverick models.

## Storage

At least 20GB–30GB free storage.

## macOS

- Ventura
- Sonoma
- Sequoia
